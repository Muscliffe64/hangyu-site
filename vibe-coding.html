写代码 Vibe Coding

一个不写代码的人，怎么靠 AI 一起把脑子里的网页做出来。这里记录我自己用 Cowork、Claude Code、Kimi、Codex 等工具做出来的小项目 —— 也是我设计的"AI 写网页"课的现场样品。

Vibe Coding 是一种"凭感觉写代码"的方式：你说你想要什么、AI 把它落到文件里。不背语法、不下载一堆开发工具，普通人也能上手。

Vibe coding — telling an AI what you want, and letting it write the file. No need to memorize syntax or set up a dev environment. Anyone can do it.

三节课的课程大纲（drafting）

  1. 01

    立骨架 · Skeleton

    学生带着自己的素材到课。一节课把网站从零搭到本地能跑 —— 有首页、有几个内容栏目、有自己的几张照片 / 一段文字。

    关键概念：HTML 是骨架 / CSS 是衣服 / JS 是动作。我们这节只碰骨架。

  2. 02

    上颜值 · Visual

    给上节课的骨架上颜色、上字体、上排版、做手机响应式。这节课结束，他们的网站在手机上看也能看。

    关键概念：怎么告诉 AI "我喜欢的感觉"。学生最容易掉进的坑是"我想要好看"，但说不出来好看是什么。

  3. 03

    推上线 · Ship

    把网站推到 Vercel。每人拿到一个真实的 URL，可以发给朋友、加在简历上。

    关键概念：什么是"部署"。为什么不用 Git 也能上线。为什么这个 URL 是免费的、可以一直用。
